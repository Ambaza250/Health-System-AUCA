# HealthSys - Student Health Record Management System

A modern, responsive **Student Health Management System** built for **Adventist University of Central Africa (AUCA)**.

## ✨ Features

- **🔐 Secure Authentication**
  - Login & Sign Up functionality
  - User session management

- **📊 Dashboard**
  - Overview with key statistics
  - Recent visits
  - Quick actions

- **📝 New Visit Recording**
  - Student information
  - Medical condition & severity
  - Treatment & prescription
  - Parent notification toggle

- **📋 All Records**
  - Full student health records table
  - Search and filter
  - **Inline editing** (Update records via modal)

- **🔍 Advanced Search**
  - Search by disease/condition
  - Visual condition cards with case counts

- **📈 Statistics & Analytics**
  - Cases per disease (dynamic bar chart)
  - Visits over time (line chart)
  - Most common conditions

- **💾 Persistent Data**
  - All data saved in browser's `localStorage`
  - No backend required

- **🎨 Modern UI**
  - Fully responsive design
  - Clean Tailwind CSS interface
  - Professional medical theme

## 🛠️ Technologies Used

- **HTML5**
- **Tailwind CSS**
- **Vanilla JavaScript**
- **Chart.js** (for analytics)
- **Font Awesome** (icons)
- **localStorage** (data persistence)
